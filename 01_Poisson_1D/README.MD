# 1D Poisson Equation ($\nu=1$)

## Strong Form

$$
\begin{aligned}
-u'' &= f &&\text{in } \mathcal{B}=(0,1), \\
u &= 0 &&\text{on } \partial\mathcal{B}_D=\{0,1\}.
\end{aligned}
$$

## Methods

**SFEM:** $(u_h',v_h')=(f,v_h)$, $u_h\in V_h\subset H^1_0$

**LSFEM:** $\mathbf{q}=u'$, $-q'=f$, $\mathcal{J}=\|q'+f\|_0^2+\|q-u'\|_0^2$
